# AGENTS

## Project Notes

- This project is a small TypeScript CLI that builds with `npm run build` and lints with `npm run lint`.
- Keep generated output and local workspace data out of version control.

## Commit Rules

- Always use Conventional Commits for every commit.
- Use commit messages in the form `type(scope): summary` when a scope is useful.
- Typical types for this project are `feat`, `fix`, `chore`, `docs`, `refactor`, and `test`.
